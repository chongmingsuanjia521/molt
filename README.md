### **Molt（蜕）项目概述**

**项目名称**：Molt（蜕）  
**中文名**：蜕  
**主题**：自律 + 社交  
**核心目标**：帮助用户建立自律习惯，并通过社交互动增强坚持动力

---

### **项目简介**
**Molt** 是一款结合 **自律管理** 和 **社交激励** 的Web与移动端应用。用户可以通过设定个人目标（如早起、健身、学习等），记录每日进展，并在社区中分享成就、互相鼓励。系统提供数据统计、成就系统、社交互动等功能，让自律变得更有趣、更可持续。

---

### **核心功能**
#### **1. 自律管理**
✅ **目标设定**：自定义目标（如“每天阅读30分钟”）  
✅ **打卡记录**：每日签到，记录完成情况  
✅ **数据统计**：可视化进度（周/月/年趋势）  
✅ **成就系统**：达成目标解锁徽章

#### **2. 社交激励**
✅ **社区动态**：分享成就、发布心得  
✅ **好友互动**：关注、点赞、评论  
✅ **挑战赛**：参与公开挑战（如“7天早起计划”）  
✅ **排行榜**：展示活跃用户与成就达人

#### **3. 管理后台**
✅ **用户管理**：审核、封禁违规用户  
✅ **内容管理**：维护社区帖子  
✅ **数据分析**：统计用户活跃度、目标完成率

---

### **技术架构**
#### **前端**
- **Web端**（Vue3 + Vite + Pinia + Element Plus）
- **移动端**（Uni-app 跨平台方案）
- **图表**（ECharts 数据可视化）

#### **后端**
- **框架**：Spring Boot + MyBatis
- **安全**：Spring Security + JWT
- **实时通知**：WebSocket
- **中间件**：Redis（缓存）、RabbitMQ（异步任务）

#### **数据库**
- **MySQL**（核心业务数据）
- **MongoDB**（日志、非结构化数据）

#### **云服务**
- **阿里云OSS**（存储用户头像、成就图片）
- **Docker + Kubernetes**（容器化部署）

---

### **开发模式**
- **敏捷开发**：分阶段交付可运行版本
- **版本控制**：Git（GitHub/GitLab）
- **每日投入**：4小时/天

---

### **项目愿景**
> “蜕”寓意破茧成蝶，象征用户通过自律实现自我提升。Molt 不仅是一个工具，更是一个陪伴用户成长的社交化自律平台。
